Requirement:

1. Python 2.7
2. PIL library
3. Fingerprint Module : https://github.com/L200160090/biometrics
4. Fingerprint image

How to:

1. Menyiapkan gambar original berformat png


2. Melakukan process poincare untuk mengetahui sudut-sudut jari
python poincare.py images/z.png 16 1 --smooth
Hasil:


3. Melakukan process gabor untuk memperjelas jalur jari tangan.
python gabor.py images/z.png 16 --save
Hasil:


4. Melakukan process thining dari gambar gabor sebelumnya untuk mempertipis jalur jari tangan.
python thining.py images/z_enhanced.gif --save
Hasil:


5. Melakukan process minutae detection dari gambar thining sebelumnya untuk mendeteksi ujung dari jalur jari.
python crossing_number.py images/z_enhanced_thinned.gif --save
