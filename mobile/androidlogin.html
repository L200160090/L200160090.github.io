Android Login dan Register menggunakan PHP MySQL

Requirement:

1. Wamp64
2. Browser
3. Android Studio
4. Koneksi wifi 1 jaringan antara android dengan pc

Langkah-langkah:
1. Jalankan program wampp
2. Pembuatan database seperti di gambar:

3. Pembuatan aplikasi di Android sesuai
https://www.androidhive.info/2012/01/android-login-and-registration-with-php-mysql-and-sqlite/
4. Lihat ip pada komputer dengan membuka cmd dan ketik "ipconfig" seperti pada bawah ini


5. Ubah config pada AppConfig.java dan sesuaikan dengan ip pada komputer


6. Klik kiri wamp64 dan klik "Put Online" digunakan untuk meng-allow koneksi dari hp menuju ke server localhost wamp


7. Edit httpd-vhost


8. Sesuaikan dengan gambar berikut, lalu pada ip sesuaikan dengan 3 angka ip pertama jaringan yg digunakan contoh : "192.168.1"


9. Compile aplikasi dan install pada handphone, lalu sambungkan dengan jaringan wifi yg sama dengan komputer.
10. Menjalankan aplikasi
